markdown语法

本篇博客是作者用于复习markdown语法的，具体详细语法参见官网教程

标题语法

标题前添加#号，#号数量代表标题的级数.

# 一级标题

一级标题

## 二级标题

二级标题

### 三级标题

三级标题

以此类推.

注意#号的后面要加一空格.

段落语法

在markdown语法中直接使用一行或多行空行来表示段落之间的分割.

换行语法

在一行的结尾添加两个空格,回车即可换行.

强调语法

粗体

在想要加粗的文本前后各加两个*号

**加粗文本**

加粗文本

斜体

在加倾斜的文本前后各加一个*号

*斜体*

斜体

斜体加粗体

在文本前后各加三个*号

***文本***

文本

引用语法

在段落前添加一个>表示应用

> 引用文本

引用文本

列表语法

有序列表

要创建有序列表，请在每个列表项前添加数字并紧跟一个英文句点.数字不必按数学顺序排列，但是列表应当以数字 1 起始.

1.First item

2.Second item

3.Third item

4.Fourth item

  1. First item
  2. second item
  3. third item
  4. fourth item

无序列表

要创建无序列表，请在每个列表项前面添加破折号 (-)、星号 (*) 或加号 (+) .缩进一个或多个列表项可创建嵌套列表.

- 无序列表

*无序列表

+无序列表

  • 无序列表
    • 无需列表
      • 无序列表

普通超链接

[链接文本](链接地址 "链接title")
